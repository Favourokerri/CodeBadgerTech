### Problem: **Find the Largest Number**

Write a program that finds and prints the largest number among five predefined integers.

#### Example:
- Given the numbers:
  - `num1 = 3`
  - `num2 = 7`
  - `num3 = 2`
  - `num4 = 9`
  - `num5 = 5`

- The output should be:
  ```
  The largest number is: 9
  ```

- Given the numbers:
  - `num1 = -1`
  - `num2 = -5`
  - `num3 = -3`
  - `num4 = -8`
  - `num5 = -2`

- The output should be:
  ```
  The largest number is: -1
  ```
